🧾 DOCUMENTACIÓN FUNCIONAL COMPLETA — IMPRIMIENDO 🧠 1. CONCEPTO GENERAL DEL SISTEMA IMPRIMIENDO es un sistema de autoservicio compuesto por: Un kiosco físico (pantalla + módulo de pago + impresora) Un sitio web móvil sin login Comunicación en tiempo real mediante sesiones temporales (QR + UUID) 🧱 2. ARQUITECTURA DE INTERFACES 🎯 Interfaces principales Interfaz Usuario Dispositivo Pantalla de negocio Cliente físico Pantalla kiosco Web móvil Cliente Celular Backend panel lógico Sistema Servidor 📺 3. PANTALLA DEL NEGOCIO (KIOSCO) 🖥️ 3.1 Pantalla Idle (Inicio) Objetivo: Atraer usuario y generar sesión Elementos visibles: Logo “IMPRIMIENDO” Texto: “Escanea el QR para comenzar” QR dinámico grande centrado Timer pequeño (opcional visual) Comportamiento: Genera automáticamente: session_id (UUID) expiration_time (40 min) Regenera QR cuando: Expira sesión Usuario termina sesión 🧾 3.2 Pantalla de Espera de Archivos Se activa cuando: Usuario escanea QR Elementos: ID de sesión visible (ej: SES-8F3A2) Estado: “Esperando archivos...” Indicador en tiempo real: “Usuario conectado” Timer restante 💰 3.3 Pantalla de Pago Se activa cuando: Usuario presiona "Imprimir $$$" desde móvil Elementos: ID de sesión Lista resumida: Archivos Número de hojas Total a pagar (MXN) Instrucción: “Inserta $XX pesos” Estado dinámico: Dinero insertado: $10 / $25 Barra de progreso de pago Botón oculto (solo backend controla flujo) 🖨️ 3.4 Pantalla de Impresión Elementos: Estado: “Imprimiendo...” Barra de progreso Archivo actual imprimiéndose Número de hojas 📥 3.5 Pantalla de Entrega Elementos: Mensaje: “Recoge tus impresiones” Animación o indicador de bandeja Botón: “Nueva sesión” (reinicia sistema) ❌ 3.6 Pantalla de Error Casos: Error de impresión Pago insuficiente Archivo inválido Elementos: Mensaje claro Código de error Botón: “Reintentar” “Cancelar sesión” 📱 4. INTERFAZ MÓVIL DEL USUARIO 🔐 4.1 Entrada por QR URL: /session/:session_id Validaciones: Sesión existe No está expirada 🧭 4.2 Dashboard de Sesión Elementos: ID de sesión Timer restante (countdown visible) Estado: “Sesión activa” 📂 4.3 Subida de Archivos Componente: Botón: “Subir PDF” Drag & Drop Lista de archivos subidos Restricciones: Solo PDF Tamaño máximo configurable ⚙️ 4.4 Configuración por Archivo Cada archivo tiene su propio panel: Opciones: Tipo: Color / Blanco y Negro Rango de páginas: Ej: 1-5, Todas Copias Orientación: Vertical / Horizontal Tamaño: Carta / Oficio Escala Doble cara (si aplica) UI: Card expandible por archivo 📊 4.5 Resumen de Impresión Elementos: Lista de archivos Configuración aplicada Precio por archivo Total acumulado 💳 4.6 Botón de Acción Botón principal: IMPRIMIR $XX MXN Acción: Envía request al backend Bloquea edición ⏳ 4.7 Estado de Pago Elementos: Mensaje: “Realiza el pago en la máquina” Estado en tiempo real: “Esperando pago...” “Pago recibido” 🖨️ 4.8 Estado de Impresión Elementos: “Imprimiendo...” Lista de progreso Archivos completados ✅ 4.9 Finalización Elementos: “Impresión completada” Botón: “Nueva impresión” Botón: “Cerrar sesión” ⚠️ 4.10 Modal de Cierre Confirmación: “¿Seguro que deseas cerrar la sesión?” Botones: Confirmar Cancelar 🔄 5. FLUJO COMPLETO DEL USUARIO Usuario ve QR en kiosco Escanea QR Se abre sesión en móvil Sube archivos Configura impresión Presiona imprimir Va a kiosco Inserta dinero Sistema valida pago Imprime Usuario recoge hojas Puede repetir o cerrar sesión 🧩 6. COMPONENTES A DESARROLLAR 📦 FRONTEND (React) Kiosco: QRGenerator SessionWatcher PaymentScreen PrintProgress ErrorScreen Móvil: SessionDashboard FileUploader FileConfigCard PrintSummary PaymentStatus PrintStatus ⚙️ BACKEND (Node + Express) Endpoints: POST /session/create GET /session/:id POST /upload POST /print/calculate POST /print/execute POST /payment/update DELETE /session/:id 🗄️ BASE DE DATOS Tabla: sessions id (UUID) created_at expires_at status Tabla: files id session_id filename config (JSON) status 🔐 7. SEGURIDAD Validación estricta de PDFs Sanitización de inputs Expiración automática Eliminación de archivos: al imprimir al expirar sesión Rate limiting por IP Headers seguros (Helmet) 🧹 8.
